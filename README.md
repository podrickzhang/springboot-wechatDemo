# sprintboot-wechatDemo
springboot做后台，微信小程序做前台的应用Demo<br>
areadisplay作为微信前端应用，springboot-wechat作为后端应用。<br>
areadisplay采用为微信小程序开发，后端应用采用springboot+mybatis+mysql开发<br>
